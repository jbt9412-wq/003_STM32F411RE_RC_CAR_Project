# STM32 FreeRTOS 자율주행 RC카

STM32F411RE와 FreeRTOS를 활용한 4WD 자율주행 및 블루투스 원격 제어 RC카 프로젝트입니다. 하드웨어 타이머를 이용해 초음파 센서를 비동기 처리하고, L298N 드라이버와 RTOS 태스크 스케줄링을 연동하여 안정적인 주행 제어를 구현했습니다.



## 주요 기능

* **FreeRTOS 기반 제어:** 센서 측정 태스크와 주행 판단 태스크를 분리하여 시스템 지연(Delay) 현상 해결
* **타이머 기반 거리 측정:** 3개의 초음파 센서 에코(Echo) 신호를 입력 캡처(Input Capture) 인터럽트로 받아, 소프트웨어 딜레이 없이 정밀하게 거리 연산
* **자율주행 모드:** 센서 값에 로우패스 필터를 적용해 노이즈를 잡고, 전방 장애물 감지 시 회피 기동(후진, 선회) 수행 및 직진 시 양측 거리 오차를 보정하여 주행
* **수동 제어 모드:** 스마트폰 블루투스 앱과 연동하여 UART 수신 인터럽트 기반 즉각 제어. 조작 버튼에서 손을 떼면 자동으로 정지하는 브레이크 기능 구현
* **차동 조향(Skid-steer):** 4개의 모터를 좌/우 2채널로 묶고, 스로틀(속도)과 스티어링(조향) 값을 믹싱하여 제어

## 하드웨어 구성

모터 구동부와 로직 제어부의 전원을 분리하여 시스템 안정성을 확보했습니다.

* **MCU:** STM32F411RET6 (Nucleo-64)
* **프레임:** 아두이노 4WD 조립 키트 (DC 기어드 모터 4개)
* **모터 드라이버:** L298N
* **전원:** 3.6V 배터리 3개 직렬(10.8V)로 모터 구동. L298N의 5V 출력을 STM32 전원으로 사용
* **센서/통신:** HC-SR04 초음파 센서 3개, HC-06 블루투스 모듈

## 핀 매핑

| 기능 | 주변장치 | 핀 | 비고 |
| --- | --- | --- | --- |
| 좌측 모터 방향 | GPIO | `PC0`, `PC1` | 전진/후진 제어 |
| 우측 모터 방향 | GPIO | `PC2`, `PC3` | 전진/후진 제어 |
| 양측 모터 속도 | TIM2 | `PA0`, `PA1` | PWM 출력 |
| 초음파 Trigger | GPIO | `PA8`, `PA9`, `PA10` | 좌/중앙/우측 |
| 초음파 Echo | TIM3 | `PC6`, `PC7`, `PC8` | 입력 캡처 |
| 블루투스 통신 | USART2 | `PA2`, `PA3` | 비동기 수신 (9600bps) |

## 소프트웨어 구조

인터럽트와 2개의 메인 태스크로 동작합니다.

1. **SensorTask:** 15ms 주기로 3개의 초음파 센서 트리거 신호를 순차 출력
2. **TIM3_IRQHandler:** 에코 신호의 에지 변화를 감지해 센서별 거리(cm) 도출
3. **UART2_IRQHandler:** 블루투스 제어 값 수신 즉시 목표 상태(모드, 방향, 속도) 갱신
4. **MotorTask:** 자율주행 모드일 때는 센서 값을 바탕으로 스스로 주행 방향을 결정하고, 수동 모드일 때는 수신된 제어 값을 그대로 L298N 드라이버에 PWM 신호로 출력

## 조작 방법

`Arduino Bluetooth Controller` 앱 설정 기준입니다.

* **모드 전환:** 자율주행(`A`), 수동 제어(`P`)
* **방향 제어:** 전진(`F`), 후진(`B`), 좌회전(`L`), 우회전(`R`), 정지(`S`) - 버튼 Release 시 `0` 전송
* **속도 제어:** 가속(`T`), 감속(`X`)
